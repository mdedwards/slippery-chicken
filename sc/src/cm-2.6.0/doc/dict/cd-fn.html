[Function]
(cd [directory])

Sets the current working directory. If directory is not specified then the working directory is set to the user's home directory.

Examples:

Example 1. Setting the current working directory. 

(cd "/tmp")
⇒ "/tmp/"
(cd)
⇒ "/Users/hkt/"
