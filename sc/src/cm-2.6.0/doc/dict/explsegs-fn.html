[Function]
(explsegs length sum power)

Like explseg but returns a list of length number of exponential segments. 

cm> (define l (explsegs 4 10 2))

cm> l
(1.8920711500272102 2.250064473703739 2.6757926813433408
  3.18207169492571)
cm> (apply #'+ l)
10.0
cm>
