[Function]
(invert list [point] [scale])

Returns the inversion of a list of notes or keynums around point in scale. Point defaults to the first element in list. 

cm> (invert '(0 1 2 3 -1))
(0 -1 -2 -3 1)
cm> (invert '(c4 d4 e4 f4 b3) 'c5)
(c4 bf4 af4 g4 cs4)
cm> (invert '(0 1 2 3 -1) 60 )
(60 71 70 69 61)
cm>
