[Class]
log-axis

Implements logarithmic axis displays for the horizontal and vertical dimensions of a plotter window. log-axis supports the following slot initializations in addition to the standard axis initializations:

:base number
An alternate name for the axis :increment initialization that makes it clear that for log-axis the increment value is used as the logarithmic base for the axis display.
