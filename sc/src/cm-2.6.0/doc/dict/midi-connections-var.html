[Variable]
*midi-connections*

Default MIDI connections used by midishare-stream if the connections are not specified when the stream is created. Connections are specied as a list: (input output) where input and output are both the string name of a Midishare client application to connect to. The default value is ("MidiShare" "MidiShare") which establishes both input and output connections directly to the main Midishare driver application. Specify false as either the input or output connection if it is is not wanted when the stream is opened.
