[Class]
midi-note-off

A MIDI note off event.

midi-note-off supports the following slot initializations:

:time number
The start time of the object.
:channel integer
A MIDI channel number. The default value is 0.
:keynum integer
A MIDI key number
:velocity integer
A MIDI velocity. The default value is 64.

Examples:

See the documentation for midi-note-on for an example of using this object.
