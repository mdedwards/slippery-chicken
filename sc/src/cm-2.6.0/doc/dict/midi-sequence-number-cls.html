[Class]
midi-sequence-number

A MIDI sequence number event.

midi-sequence-number supports the following slot initializations:

:time number
The start time of the object.
:number integer
The MIDI sequence number.

Examples:

Example 1. The midi-sequence-number event. 

(new midi-sequence-number :time 0 :number 1)
