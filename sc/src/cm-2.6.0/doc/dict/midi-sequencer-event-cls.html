[Class]
midi-sequencer-event

A MIDI sequencer event.

midi-time-signature supports the following slot initializations:

:time number
The start time of the object.
:data list
A list of data bytes for the sequencer. The first byte (or first three bytes if the first byte is 0) specifies the manufacturer's ID and the following bytes contain information specific to the manufacturer.

Examples:

None.
