[Class]
midi-text-event

A MIDI text event.

midi-text-event supports the following slot initializations:

:time number
The start time of the object.
:type integer
The "text type" of the message, an integer 1-7 corresponding to one of the following conditions: (1) text, (2) copyright notice, (3) sequence or track name, (4) instrument name, (5) lyric, (6) marker, (7) cue point.
:text string
The text (string) of the message.

Examples:

Example 1. The midi-text-event event. 

(new midi-text-event :time 0 :type 1 :text "HiHo!")

(new midi-text-event :time 0 :type 2 :text "Gnu subversion")
