[Macro]
(new class {keyword value}*)

Allocates and initalizes a new object of type class, an unquoted class name (symbol). Following class comes zero or more keyword arguments where each keyword initializes a slot in the new object with its value.

Examples:

Example 1. Creating objects with new. 

(new seq :name 'test)
⇒ #<seq: "test">
(new midi :keynum (between 32 89) :time 1 :duration 3)
⇒ #i(midi time 1 duration 3 keynum 53 amplitude 64 channel 0)
(new heap :rhythms '(q w e h e.. ))
⇒ #<heap @ x4C357A6>
