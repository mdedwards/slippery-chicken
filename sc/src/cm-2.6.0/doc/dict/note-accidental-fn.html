[Function]
(note-accidental note)

Returns the accidental of a note or note class.

Examples:

Example 1. Accessing note accidentals. 


(note-accidental 'dff5)
⇒ ff
(note-accidental 'd5)
⇒ #f
(note-accidental 'dn5)
⇒ n
(note-accidental 'ds)
⇒ s
