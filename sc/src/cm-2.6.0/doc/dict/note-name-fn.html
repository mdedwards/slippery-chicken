[Function]
(note-name note [scale])

Returns the name of note with octave number removed.

Examples:

Example 1. Returning the note name of a note. 

(note-name 'df5)
⇒ df
(note-name 'dn5)
⇒ dn
(note-name 'd5)
⇒ d
(note-name 'd)
⇒ d
