[Function]
(plotter-front-styling plotter :view :color :point-width :point-height :line-width :font :redraw)

Returns a list of the styling properties of the front layer of plotter. Styling properties can be changed by specifying their new value using any of the keyword arguments which are identical to the keywords documented under plotter arguments.
