[Function]
(plotter-property plotter property . value [:redraw bool])

Returns the value of the specified "window property" property in plotter. If value is specified then property is set to that value first. If :redraw is true (the default) then the window is redrawn if a property value changes.

plotter-property recognizes the following window properties:

:title string
The title of the window.
:show-grid boolean
If true the background plotting grid is displayed, otherwise it is not.
:show-back-layers boolean
If true the background layers are displayed, otherwise they are not.
:show-guide boolean
If true the mouse editing guide is displayed, otherwise it is not.
