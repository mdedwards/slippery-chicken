[Function]
(rescale number old-min old-max new-min new-max)

Returns a value between new-min and new-max that is proportional to number between old-min and old-max. 

cm> (rescale .51 0 1 0 100)
51.0
cm> (rescale 1.2 1 2 80 90)
85.0
cm>
