[Function]
(scale-min freq1 freq2 [scale])

Returns freq2 if it is less than freq1 in scale, otherwise returns freq1. 

cm> (scale-min 60 'c4)
60
cm> (scale-min 'css4 'cs4)
cs4
cm>
