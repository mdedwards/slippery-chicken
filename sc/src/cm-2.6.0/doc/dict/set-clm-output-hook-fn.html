[Function]
(set-clm-output-hook! function)

Causes function to be called on clm files after they are written by events. The function is passed the pathname (string) of the file and any initialization arguments the user set in the file. These additional arguments are passed as keyword and value pairs to the hook function. To remove an existing output hook specify function as false.
